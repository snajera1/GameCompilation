# Quick 'n Fun Game Compilation
CSI2300 Final Project

Title: Quick 'n Fun Game Compilation
Team Name: Quick Connections
Member: Samuel Najera

The main idea for this project is to build an app with multiple quick and fun games for someone to play in order to kill some time, or if they want something short to do with a friend. Currently, the games I have planned for this are tic-tac-toe, hangman, and connect four. I'm aiming to allow the player to choose between playing against an opponent or the computer and will keep track of each player's win count. I'm building this because as a college student, I don't have a lot of sustained downtime, but I have a lot of smaller pockets of free time, where I don't have time to do much of anything substantial, but it still feels like a waste of time just to spend a few minutes scrolling through social media on my phone. I also run into friends on campus in between classes and only have a little bit of time to catch up with them, and I think a quick and simple game compilation could be a fun way to connect with them better in that short window.

I plan to put a lot of effort into this program. I don't think the actual programming of the games will be all that difficult, but I think it will be a big challenge fitting every game into a shell that maximizes convenience and efficiency, as well as implementing the extra features (like the win tracker and the AI opponent for singleplayer).
